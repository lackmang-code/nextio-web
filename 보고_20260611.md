### 2. 생성기 손상 복구 (중요)
- `make_promo.py`·`make_index.py` 두 파일이 **오늘 13:13:00에 외부에서 손상**되어 있었음
- 증상: 모바일 반응형 `@media` 블록이 docstring 자리(파일 맨 앞)에 잘못 삽입 → Python SyntaxError로 **실행 불가**
- 패턴: @media가 세 군데 삽입 — ① docstring 앞(오염), ② docstring 안 중복(오염), ③ CSS 문자열 안(올바른 위치)
- 복구: ①②의 오염만 제거, ③ CSS 안의 모바일 @media는 **모바일 반응형 의도이므로 보존**
- 결과: 두 생성기 정상화 → 카드/목차 정상 생성 확인
