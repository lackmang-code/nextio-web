1단계: 타겟이 뭔데?
👤
박낭만

에이전트야, 헬스케어 콘텐츠 에이전시 창업하려는데 "타겟 시장 분석"을 해야 한다는데 이게 뭐야? 어떻게 해?

🤖
에이전트(영업팀)

좋은 시작이에요. 타겟 시장 분석이라고 하면 어렵게 들리는데, 핵심은 딱 하나예요:

지금 박낭만 씨 상황으로 바꾸면: "헬스케어 관련 뉴스레터·매거진을 살 수 있는 기관이 세상에 몇 종류나 있고, 그중 어디가 가장 살 것 같은가?" 를 정리하는 거예요.

이걸 왜 해야 하냐면 — 세일즈를 시작할 때 "아무한테나 연락하기"랑 "가장 살 것 같은 곳부터 연락하기"는 성과가 완전히 달라요. 지도를 그려야 어디서 시작할지 보입니다.

같이 해볼게요. 먼저 — 헬스케어 쪽에서 뉴스레터나 매거진을 낼 만한 기관이 어떤 종류가 있을까요? 아는 대로 말해보세요.

👤
박낭만

음... 병원이랑 제약회사? 의료기기 회사? 그리고 의학회 같은 것도 있나?

🤖
에이전트(영업팀)

잘 나왔어요. 근데 박낭만 씨가 10년 현장에 있었으면 더 보일 거예요. 제가 힌트 드릴게요.

병원을 더 쪼개면:

  • 대학병원 (이미 자체 팀 있음 → 경쟁 어려움)
  • 전문병원 (척추·관절·안과 등) → 여기가 기회
  • 의원급 (동네 병원 체인)

제약·의기 쪽도:

  • 대형 제약사 (MSD, 화이자 → 이미 자체 팀)
  • 중소 제약·바이오벤처 → 여기가 기회

그리고 박낭만 씨가 놓친 게 있어요:

  • 의학회·학술단체 (협회)
  • 병원협회 (지역 단위)
  • 의료기기 협회
  • 헬스케어 스타트업 지원기관

에이전트가 사용자의 경험을 끌어내는 질문을 먼저 하고, 빠진 부분을 채워주는 방식이다. 비전문가는 자기가 가진 지식을 "정리"하는 것만으로도 큰 발견을 한다. 에이전트가 정답을 말하기 전에 사람이 먼저 생각하게 유도하는 것이 교육 효과가 높다.

2단계: 분류 기준 만들기
👤
박낭만
🤖
에이전트(영업팀)

헬스케어 콘텐츠 에이전시 기준으로 이렇게 나눌 수 있어요:

  1. 전문병원 (척추·관절·안과·피부·성형 등)
  2. 중소 종합병원 (지역 300병상 이하)
  3. 대학병원 학과·진료팀 (병원 전체 아닌 과 단위)
  4. 의학회·학술단체 (각과 학회, 연구회)
  5. 중소 제약·바이오벤처
  6. 의료기기 중소기업
  7. 헬스케어 스타트업·디지털헬스 기업
  8. 병원 창업 지원기관·의료 액셀러레이터

이렇게 8가지요.

근데 이걸 나눈 것만으로는 부족해요. "어디가 더 살 것 같나"를 알아야 하거든요. 그래서 각 유형마다 "지금 이미 정기 발행을 하고 있나?"를 기준으로 등급을 매겨요.

👤
박낭만

등급을 어떻게 매겨?

🤖
에이전트(영업팀)

간단해요. 3가지 등급이에요:

🟢 A등급: 정기 매거진·뉴스레터가 아직 없음 → 빈자리 → 우리 1순위 타겟
🟡 B등급: SNS·카드뉴스는 있는데 정기 발행물은 없음 → 업그레이드 제안 가능 → 2순위
🔴 C등급: 이미 자체 팀이나 외주로 정기 발행 중 → 진입 어려움 → 지금은 패스

이 등급 기준을 먼저 문서로 만들어두면, 나중에 새 기관 조사할 때마다 바로 분류할 수 있어요.

시장지도 등급 기준

> 조사 시 이 기준으로 각 기관에 등급 부여

등급 정의

등급의미영업 전략
🟢 A정기 매거진·뉴스레터 없음 → 완전한 빈자리1순위 공략 — "새로 시작하면 어떨까요?"
🟡 BSNS·카드뉴스는 있지만 정기 발행물 없음 → 업그레이드 여지2순위 공략 — "있는 소재를 매거진으로 묶어요"
🔴 C이미 정기 발행 중 (자체 팀 또는 외주) → 진입 어려움보류 — 현재 공급자 교체 기회 있을 때만
⬜ 미확인정보 불충분 → 콜드콜로 확인 필요전화 1통으로 A/B/C 결정 가능

조사 방법 (4단계)

  1. 구글 검색: 기관명 + "뉴스레터" 또는 기관명 + "매거진"
  2. 공식 홈페이지: 소식/발행물 메뉴 확인
  3. SNS 확인: 인스타·유튜브 계정 (정기 콘텐츠 여부)
  4. 스티비 검색: stibee.com에서 기관명 검색 (레터 발행 여부)

등급 판단 기준

  • 위 4단계 검색에서 정기 발행물 발견 안 됨
  • 홈페이지에 뉴스레터·소식지 메뉴 없음
  • 인스타·유튜브 등 SNS는 활발하게 운영 중
  • 카드뉴스·단편 영상은 있음
  • 하지만 정기 구독·발송 형태의 뉴스레터/매거진은 없음
  • 정기 발행물 홈페이지 메뉴 확인됨
  • 구독 신청 링크 있음
  • 스티비 등 뉴스레터 플랫폼 사용 확인

#유형구체적 대상특징접근 키워드
1전문병원척추·관절·안과·피부·성형 전문병원브랜드 차별화 필요, 환자 신뢰 구축 수요"환자 신뢰", "진료 철학 전달"

4의학회·학술단체각 의학 분과 학회, 연구회회원 소통, 학술 정보 공유"회원 커뮤니케이션", "학술 이슈 정리"

6의료기기 중소기업국내 의료기기 제조·유통 중소기업의사·병원 대상 B2B 홍보, 해외 진출 지원 수요"의료진 대상 정보", "제품 신뢰도 구축"
7헬스케어 스타트업디지털헬스, 원격진료, 웰니스 앱투자자·소비자 동시 소통 필요"비전 공유", "유저 스토리"
8지원기관·의료 액셀러레이터의료기기 창업지원, 헬스케어 AC입주 기업 홍보 대행, 기관 브랜딩"스타트업 성과 홍보", "생태계 소식"

3단계: 실제 조사 시작

박낭만

근데 이걸 어떻게 조사해? 일일이 홈페이지 다 들어가봐야 해?

🤖
에이전트(영업팀)

그게 현실이에요. 근데 효율적으로 하는 방법이 있어요.

조사 순서:

  1. 기관명 + "뉴스레터" 구글 검색
  2. 기관 공식 홈페이지 직접 확인 (발행물·소식 메뉴)
  3. 인스타그램·유튜브 계정 확인 (정기 콘텐츠 있는지)
  4. 스티비(stibee.com) 검색 (많은 기관이 여기서 레터 발행)

이 4단계면 10분 안에 한 기관 파악돼요.

직접 몇 개 해볼까요? 제가 같이 조사해드릴게요.

👤
박낭만

응, 해봐줘.

🤖
에이전트(영업팀)

좋아요. 유형별로 대표적인 곳들 샘플 조사해볼게요.

  • 세브란스 → 자체 매거진 여럿 → 🔴C
  • 힘찬병원 → 유튜브·방송 중심, 정기 웹진 없음 → 🟡B
  • 지역 척추 전문병원들 → 대부분 SNS만 → 🟡B~🟢A 많음
  • 대형 제약(유한양행, 동아) → 자체 홍보팀, 콘텐츠 다 있음 → 🔴C
  • 50인 이하 바이오벤처 → 블로그·카드뉴스만 → 🟡B 대부분

규모가 클수록 C등급 (이미 다 함), 규모가 작을수록 A·B등급 (아직 안 함)이에요.

이 패턴이 "우리 타겟은 대형 기관이 아니라 중소·전문 단위" 라는 핵심 전략을 알려줘요.

🎓 강사 노트 ▼ 펼치기

"시장 조사"는 결국 패턴 발견이다. 에이전트가 개별 결과를 나열하는 게 아니라 "패턴이 보이시나요?" 하고 인사이트를 뽑아주는 것이 핵심이다.

  • 척추·관절·안과 전문병원 (지역 중소 규모)

의료기기 회사 10년 경력이 있잖아요. 그러면 의료기기 중소기업 사보/뉴스레터 는 "나 이 업종 알아요"라는 신뢰가 바로 생겨요. 첫 고객을 여기서 잡으면 레퍼런스가 생기고, 그다음 제약·병원으로 확장하기 쉬워요.

👤
박낭만

오, 그 생각을 못 했네. 나한테 유리한 게 있었구나.

맞아요. 이게 시장지도의 진짜 가치예요. 단순히 "누가 있나" 가 아니라 "내가 가장 이길 수 있는 곳이 어디인가"를 찾는 거거든요.

이 지도는 한 번 만들면 끝이야, 아니면 계속 업데이트해야 해?

🤖
에이전트(영업팀)

계속 업데이트해야 해요. 이유 세 가지:

  1. 새 기관 발굴 — 처음엔 다 못 조사함. 영업하면서 알게 되는 곳 계속 추가
  2. 등급 변동 — B였던 곳이 갑자기 자체 뉴스레터 시작하면 C로 내려감. 반대로 포기하면 다시 A가 되기도 함
  3. 계절성 — 의학회는 학회 전후로 콘텐츠 수요가 급증. 그 타이밍을 지도에 메모해두면 나중에 영업 타이밍이 보임

주기는 한 달에 한 번 가볍게 확인하면 충분해요.
