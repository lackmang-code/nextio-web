견적서 이미 만들었는데, 산출 근거 따로 정리해야 해?

🤖
에이전트(영업팀)

역할이 달라요.

견적서산출 근거
독자고객나(대표)
목적"얼마냐" 전달"왜 이 가격이냐" 검증
포함 내용최종 가격·항목시장가 비교 + 원가 계산
공유 여부고객에게 보냄내부 보관

고객한테는 견적서만 보내고, 산출 근거는 내가 "이 가격이 맞나?" 자기 검증에 쓰는 거예요.

👤
박낭만

그러면 산출 근거를 왜 만들어?

🤖
에이전트(영업팀)
  1. 가격 협상 시: "왜 이 가격이냐"는 질문이 왔을 때 근거가 있어야 흔들리지 않음
  2. 손해 방지: "내가 손해 보는 가격인지" 계산 안 하면 일할수록 적자
  3. 단가 업데이트 근거: 6개월 후 가격 올릴 때 "시장가가 올랐다"는 근거가 있어야
🎓 강사 노트 ▼ 펼치기

인쇄소 영업 출신은 "단가표 = 절대기준"으로 생각하는 경향이 있다. 산출 근거가 단가표를 만드는 과정임을 강조.

2단계: 두 가지 방법 — 위에서 아래로 vs 아래에서 위로
👤
박낭만

산출 근거는 어떻게 만들어?

🤖
에이전트(영업팀)

두 가지 방법을 같이 쓰면 돼요:

시장에서 비슷한 서비스가 얼마에 팔리나 → 그 구간에 내 가격이 맞는지 확인

예: 콘텐츠 마케팅 에이전시 월 대행료 = 500만~1,500만
→ 내 가격 800만이면 중단 → 말이 됨

내가 이 일 하는데 실제로 얼마나 드나 → 원가 합산 → 마진 더하기

예: 원고 5만 + 디자인 15만 + 기획 10만 = 직접원가 30만
→ 마진 40% 더하면 → 가격 50만

  • Top-down = 인쇄소 주변 경쟁사 단가 확인 (시장가 파악)
  • Bottom-up = 종이값 + 잉크값 + 인건비 계산 (원가 계산)
  • Top-down만: 시장가 맞는데 내가 적자일 수 있음
  • Bottom-up만: 원가 맞는데 시장가 너무 낮거나 높을 수 있음

→ 두 결과가 비슷하면 "이 가격이 맞다"고 확신할 수 있어요

3단계: Top-down 실습
👤
박낭만

Top-down 어떻게 해?

🤖
에이전트(영업팀)

인터넷에서 비슷한 서비스 3~5개 가격 찾기예요.

"편집디자인 + 인쇄 발주 대행"의 시장가를 찾아볼게요.

찾을 키워드:

  • "편집디자인 가격"
  • "디자인 단가표"
  • "출판편집 외주 가격"

제가 알고 있는 시장가:

항목시장 단가출처
편집디자인 (페이지당)1만~3만 원디자인 견적 사이트
브로슈어·카탈로그 (기획포함)페이지당 5만~15만편집 전문 플랫폼
원고 교정·윤문꼭지당 2만 원뉴스레터 플랫폼
👤
박낭만

근데 나는 디자이너도 고용하고, 외주도 주고. 그게 섞이는데?

🤖
에이전트(영업팀)

그래서 Bottom-up이 필요해요. 네가 실제로 뭘 하든, 그 비용을 쌓는 거예요.

4단계: Bottom-up 원가 적산
👤
박낭만

Bottom-up은 어떻게 해?

🤖
에이전트(영업팀)

이 서비스 하는 데 드는 비용을 항목별로 쌓아요.

항목설명단가 기준
원고 제작·교정꼭지당 시간 × 시급시장 외주단가 기준
디자인·편집페이지당외주 단가 or 내 시급
사진·이미지스톡 구매 or 촬영실비
기획·PM시간 × 시급고정비 배분
배포·QA시간 × 시급고정비 배분

```
원고 교정 5꼭지 × 2만 = 10만
편집디자인 10페이지 × 2만 = 20만
기획·PM = 10만
배포·검수 = 5만
─────────────────────────
직접원가 합계 ≈ 45만
```

👤
박낭만

그러면 나는 얼마에 팔아야 해?

🤖
에이전트(영업팀)

직접원가에 마진을 더하면 돼요.

5단계: 매출총이익률 — 마진이 얼마나 남아야 해?
👤
박낭만

마진이 몇 %야 해?

🤖
에이전트(영업팀)

```
매출총이익률 = (가격 - 직접원가) ÷ 가격 × 100

예: 가격 65만, 직접원가 45만
→ (65만 - 45만) ÷ 65만 × 100 = 30.7% ✅
```

  • 직접원가에는 사무실·전기·통신·영업비가 안 들어감
  • 이 간접비를 빼면 순이익이 낮아짐
  • 30% 미만이면 힘들게 일하고 남는 게 없음
상황최소 마진
직접원가만 계산한 경우30% 이상
간접비까지 합산한 경우15~20% 이상

인쇄소에서는 기계 감가상각·재료비가 원가에 들어가는데 — 서비스업은 사람 시간이 핵심 원가예요.

👤
박낭만

직접원가가 45만이면 가격이 65만이 되네?

🤖
에이전트(영업팀)

맞아요. 역산하면 이래요:

```
목표 매출총이익률 = 30%
직접원가 = 45만

가격 = 직접원가 ÷ (1 - 목표이익률)
= 45만 ÷ 0.7
= 64.3만 → 올림해서 65만 또는 70만
```

6단계: 두 방법 교차검증
👤
박낭만

Top-down이랑 Bottom-up 결과가 다르면?

🤖
에이전트(영업팀)

교차검증하는 거예요.

결과의미처리
두 결과가 비슷가격이 합리적그 가격으로 확정
Bottom-up > Top-down내 원가가 시장가보다 비쌈원가 절감 방법 찾기
Top-down > Bottom-up시장보다 싸게 받고 있음가격 올릴 여지 있음

박낭만 씨 예시:

  • Top-down: 비슷한 서비스 시장가 60만~80만
  • Bottom-up: 원가 45만, 마진 30% 적용 → 65만
  • 결과: 65만 ✅ 두 방법 일치 → 가격 확정

방법 1: Top-down (시장가 비교)

비슷한 서비스가 시장에서 얼마에 팔리는지 확인

  1. 내 서비스와 비슷한 서비스 3~5개 검색
  2. 가격 범위 파악 (최저~최고)
  3. 내 가격이 그 범위 안에 있는지 확인
  • "편집디자인 가격"
  • "콘텐츠 마케팅 대행 월 비용"
  • "뉴스레터 제작 서비스 가격"
항목시장 단가
편집디자인 (페이지당)1만~3만 원
브로슈어·카탈로그 (기획+편집)페이지당 5만~15만
원고 교정·윤문꼭지당 2만 원
취재 기사 원고료200자 원고지 매당 1만~1.5만
콘텐츠 마케팅 에이전시 월 대행500만~1,500만
내 가격 위치의미
하단 (최저 근처)진입형·경쟁력 강, 마진 낮을 수 있음
중단표준 시장 가격
상단프리미엄 포지셔닝, 차별화 근거 필요

방법 2: Bottom-up (원가 적산)

내가 이 일을 하는 데 실제로 드는 비용 합산

항목계산 방법예시 단가
원고 집필·취재꼭지 수 × 평균 원고 매수 × 매당 단가매당 1만~1.4만
원고 교정·윤문꼭지 수 × 꼭지당 단가꼭지당 2만
디자인·편집페이지 수 × 페이지당 단가페이지당 1만~3만
사진·이미지실비스톡 20만 / 촬영 60만
기획·PM시간 × 시급건당 10만~40만
배포·QA시간 × 시급건당 5만~20만

```
직접원가 = 집필비 + 디자인비 + 이미지비 + 기획비 + 배포·QA비
```

교차검증

상황의미처리
Top-down ≈ Bottom-up가격이 합리적확정
Bottom-up > Top-down원가가 시장보다 비쌈원가 절감 방법 찾기
Top-down > Bottom-up시장보다 싸게 받는 중가격 올릴 여지 있음

```
Top-down: 시장가 범위 60만~80만
Bottom-up: 직접원가 45만 + 마진 30% → 65만

결과: 65만 → 시장가 범위(60~80만) 안에 있음 ✅
```

사용 방법

합산 및 가격 산출

```
직접원가 합계 = 콘텐츠 소계 + 디자인 소계 + 이미지 소계 + 기획·운영 소계
= __________ 만 원

목표 매출총이익률 = 30%

적정 가격 = 직접원가 ÷ 0.70
= __________ ÷ 0.70
= __________ 만 원

→ 올림하여 최종 가격 결정: __________ 만 원

최종 매출총이익률 확인:
= (최종 가격 - 직접원가) ÷ 최종 가격 × 100
= __________ % (30% 이상인지 확인)
```
