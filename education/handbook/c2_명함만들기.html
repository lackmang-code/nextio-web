1단계: 인쇄소에 처음 연락하는 법
👤
박낭만

에이전트야, 인쇄소에 어떻게 연락해? 전화해야 해?

🤖
에이전트(총무팀)

온라인 인쇄소는 전화 필요 없어요. 웹사이트에서 다 처리해요.

두 가지 루트:

  • 사이트 접속 → 사양 선택 → PDF 업로드 → 결제
  • 레드프린팅, 프린트링, 인쇄24, 성원애드피아 등
  • 명함 100장에 3~5만원, 3~5일 배송
  • 맞춤 상담, 색교정, 고급 용지·후가공 상담 가능
  • 처음 발주거나 색상이 중요한 경우

박낭만 씨는 어떤 용도예요? 단순 첫 발주면 온라인, 색상 정확도가 중요하면 오프라인 고려.

👤
박낭만

색상이 중요해. 브랜드 컬러가 정확하게 나와야 하거든.

🤖
에이전트(총무팀)

그러면 이메일로 견적 요청 + 색교정 요청이 맞아요.

인쇄소에 이메일로 이걸 보내면 돼요:

  1. 명함 사양 (규격·수량·용지·후가공)
  2. 인쇄용 PDF 파일 첨부
  3. 색교정 가능 여부 문의
  4. 견적 및 납기 요청

이 내용을 담은 견적 요청 이메일을 같이 만들어볼게요.

🎓 강사 노트 ▼ 펼치기

콘텐츠 기획자 출신은 "이메일로 커뮤니케이션"이 익숙하다. 인쇄소와의 첫 접촉도 이메일로 한다는 것을 자연스럽게 연결하는 것이 포인트다.

2단계: 견적 요청 이메일 작성
👤
박낭만

이메일에 뭘 써야 해?

🤖
에이전트(총무팀)

견적 요청 이메일 구성:

제목: [회사명] 명함 인쇄 견적 요청

본문 항목:

  1. 규격 (90x50mm, 양면, 4도 컬러)
  2. 용지 요청 또는 추천 요청
  3. 후가공 (무광 코팅 등)
  4. 수량 (100/200/500매 구간 단가 요청)
  5. 색교정 가능 여부 및 비용
  6. 납기 일정
  7. 파일 형식 (PDF, 블리드 3mm, 재단선 포함)

마지막: 연락처 + 빠른 회신 부탁드립니다

👤
박낭만

파일도 같이 보내야 해?

🤖
에이전트(총무팀)

첫 견적 요청 단계에서는 파일 안 보내도 돼요.
사양만 보내서 견적 먼저 받고 - 확정 후 파일 전달.

이유: 여러 인쇄소에 동시에 견적을 요청할 때 파일을 매번 보내면 번거로워요.
사양서만 보내고, 선정 후 파일 전달.

단, 색교정 요청 시에는 파일 첨부 필요. 파일 없으면 색교정 불가.

3단계: 여러 곳에 비교 견적
👤
박낭만

한 곳에만 보내야 해?

🤖
에이전트(총무팀)

아니요. 3곳에 동시에 보내는 게 표준이에요. "비교 견적"이라고 해요.

왜 3곳?

  • 가격 차이가 의외로 크게 날 수 있어요
  • 납기/색교정 서비스도 업체마다 달라요
  • 한 곳만 보면 비싼지 싼지 판단 불가

콘텐츠 기획 외주 줄 때 여러 업체 받아보는 거랑 같아요.

비교할 항목:

  • 수량별 단가
  • 색교정 가능 여부 및 비용
  • 납기 일정
  • 배송비
  • 용지/후가공 옵션
👤
박낭만
🤖
에이전트(총무팀)

단순히 제일 싼 곳 고르면 안 돼요. 판단 기준:

기준고려사항
색교정 지원 여부브랜드 컬러 중요하면 최우선 기준
납기언제까지 필요한지 역산
용지 품질샘플 요청 또는 후기 확인
가격3개 중 중간 정도가 안전 (최저가는 품질 우려)
커뮤니케이션회신이 빠르고 친절한 곳이 이후 수정도 편함

첫 발주는 소량으로 시작해서 - 결과 확인 후 주거래 인쇄소로 정하는 게 좋아요.

4단계: 수량 결정
👤
박낭만

처음에 몇 장 해야 해?

🤖
에이전트(총무팀)

이유:

  • 첫 명함은 정보 바뀔 가능성 높아요 (도메인 추가, 전화번호, 직함 변경 등)
  • 오탈자 발견 → 다 버려야 함
  • 200장이면 1~2년치 충분 (미팅 1건당 1~3장 배포 기준)
상황권장 수량
첫 발주 (정보 바뀔 수도 있음)100~200장
정보 확정, 행사 많음300~500장
대량 영업, 전시회 참가500~1,000장

단가는 많을수록 낮아지지만 - 재고 쌓이면 바꾸기 힘들어요.

👤
박낭만

도메인이 아직 확정 안 됐으면 어떡해?

🤖
에이전트(총무팀)

두 가지 방법:

방법 1: 일단 도메인 없이 발주

  • 이름/전화/이메일만 - 나중에 도메인 확정 후 재발주

방법 2: 도메인 결정 후 발주

  • 2~4주 기다리더라도 완전한 명함으로 시작

전문직 명함이면 도메인 있는 게 훨씬 나아요. 잠깐 기다리는 게 낫습니다.

5단계: 파일 전달과 색교정
👤
박낭만

인쇄소 선정하면 그다음은?

🤖
에이전트(총무팀)

인쇄소 선정 후 단계:

  1. 파일 전달

- 인쇄용 PDF (블리드 3mm, 재단선 포함, CMYK, 300dpi)
- 이메일 첨부 또는 인쇄소 웹사이트에 업로드

  1. 색교정 진행 (선택)

- 인쇄소가 출력 샘플 1장 보내줌
- "이 색으로 나올 거예요" 확인용
- 추가 비용 발생 (보통 5,000~20,000원)
- 브랜드 컬러 중요하면 필수

  1. 색교정 승인 → 본인쇄

- "이 색으로 진행하세요" 확인 답장

  1. 배송

- 3~7일 후 명함 도착

👤
박낭만

색교정 1회면 돼?

🤖
에이전트(총무팀)

보통 1회로 돼요. 다만:

  • 색교정 보고 "너무 다르다" → 수정 요청 가능 (추가 비용/납기 연장)
  • 처음부터 CMYK 변환 잘 되어 있으면 색교정에서 크게 다르게 안 나와요
🎓 강사 노트 ▼ 펼치기

"처음 인쇄 발주는 미지의 세계"처럼 느끼는 수강생이 많다. 에이전트가 단계를 명확하게 나눠주면 "이 순서대로 하면 되는구나"를 알게 된다. 특히 색교정 단계는 많은 사람이 모르고 지나치는 부분이다.

6단계: 발주 전 최종 체크
👤
박낭만

발주 버튼 누르기 전에 마지막으로 뭘 확인해야 해?

🤖
에이전트(총무팀)

발주 직전 5가지:

  1. 오탈자 - 이름/이메일/전화 다시 한 번
  2. QR 스캔 테스트 - 실제 폰으로 다시 확인
  3. 블리드 3mm 포함 확인 - 디자이너에게 최종 확인
  4. 수량 결정 - 지금 확정 못 하면 발주 미루기
  5. 배송 주소 - 정확하게 입력

이 5개 다 되면 - 발주해도 돼요.

박낭만

명함 하나 만드는 데 이렇게 많은 단계가 있는 줄 몰랐어.

🤖
에이전트(총무팀)

콘텐츠 기획도 비슷해요. 한 편의 기사가 기획/취재/초안/편집/발행 여러 단계인 것처럼.
명함도 사양/QR/파일/발주/색교정/수령의 과정이에요. 미리 알면 당황하지 않아요.

> 발주 버튼 누르기 전 이 항목 모두 확인

A. 정보 최종 확인

#항목완료
A1이름 오탈자 없는지☐
A2이메일 정확한지 (전송 테스트 권장)☐
A3전화번호 정확한지☐
A4직함·자격명 정확한지☐
A5홈페이지 주소 실제 접속되는지☐

B. QR 코드

#항목완료
B1아이폰으로 스캔 → 연락처 저장 확인☐
B2안드로이드로 스캔 → 연락처 저장 확인☐
B3저장된 연락처 정보 정확한지☐

C. 인쇄 파일

#항목완료
C1블리드(재단여백) 3mm 포함됐는지☐
C2재단선 포함됐는지☐
C3CMYK 모드인지 (또는 인쇄소가 변환해주는지)☐
C4해상도 300dpi 이상인지☐
C5폰트 아웃라인 처리 또는 임베딩됐는지☐

D. 발주 조건

#항목완료
D1수량 결정 완료☐
D2용지·후가공 선택 완료☐
D3색교정 진행 여부 결정☐
D4납기 날짜 확인 (필요 날짜보다 여유 있는지)☐
D5배송 주소 정확한지☐
D6결제 수단 준비됐는지☐

E. 발주 후 할 일

#항목
E1주문 확인 이메일 저장
E2색교정 샘플 수령 → 색상 승인
E3수령 후 QR 스캔 테스트 (인쇄본)
E4오탈자 없는지 최종 확인
E5이상 없으면 → 보관 + 영업 활동 시작

발주 미루는 게 나은 상황

  • 도메인/이메일 아직 미확정
  • QR 스캔 테스트 미실시
  • 브랜드 컬러 HEX 값 미확정
  • 디자인 파일 블리드 미포함

위 상황이면 발주 미루고 해결 후 진행. 성급한 발주 = 재발주 비용 발생.
