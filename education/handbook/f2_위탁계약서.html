  • "얼마 준다고 했잖아요" vs "그런 말 안 했어요" → 분쟁 발생

IP 귀속 조항 가이드

> 결과물의 권리는 누가 갖는가 — 콘텐츠 사업 핵심 조항

IP (지적재산권)란?

  • 저작권: 창작물 (글, 이미지, 디자인, 영상)의 권리
  • 상표권: 브랜드명·로고
  • 영업비밀: 방법론·노하우

콘텐츠 사업에서는 저작권이 핵심

3가지 귀속 방식

방식내용콘텐츠 사업 적합도
고객 귀속대금 완납 후 결과물 권리 이전✅ 일반적
공동 소유양측 모두 소유파트너십 협업 시
라이선스 방식권리는 내가 갖고 고객에게 이용 허락방법론·템플릿 포함 시

권장 조항 (콘텐츠 위탁 제작)

> "본 계약에 따라 제작된 콘텐츠의 저작권은 대금 완납이 확인된 시점에 갑에게 귀속된다."

> "단, 을이 계약 수행에 활용한 편집 방법론, 레이아웃 템플릿, 자체 제작 도구 및 소프트웨어는 을의 독점 소유이며 이전 대상에서 제외된다."

"대금 완납 후" 이전이 중요한 이유

```
대금 완납 전 이전:
┌─────────────────┐
│ 고객이 결과물 수령 │
│ → 대금 안 줘도 됨 │
│ → 내가 아무 레버리지 없음 │
└─────────────────┘

대금 완납 후 이전:
┌─────────────────┐
│ 대금을 줘야 │
│ → 결과물 권리를 받음 │
│ → 내가 레버리지 있음 │
└─────────────────┘
```

자주 발생하는 분쟁 유형

분쟁예방 조항
"이 디자인 스타일을 다른 고객에도 쓰면 안 된다""방법론·템플릿은 을 소유" 명시
"계약 기간 외에도 무상 수정 요구"수정 횟수 상한 명시
"원본 파일 다 내놔라""납품 파일 형식" 범위 명시
"다른 사람한테 똑같이 해줬잖아요"독점 제공 조건 협의 시에만 명시

독점 제공 조항 (선택)

고객이 "우리만 이 방식 쓰게 해달라"고 요구하는 경우:

> "을은 본 계약 기간 중 [지역/업종] 내 다른 고객에게 동일한 서비스를 제공하지 않는다."

  • 독점 제공은 영업 기회 제한이므로 비용 추가 필수
  • 통상 20~50% 프리미엄

체크리스트

#항목완료
1결과물 권리 귀속 방식 결정됐는가☐
2"대금 완납 후 이전" 문구가 있는가☐
3방법론·템플릿 분리 조항이 있는가☐
4납품 파일 형식·범위가 명시됐는가☐
5독점 조항 필요 여부 확인됐는가☐

착수금이 필수인 이유

착수금 없이 작업을 시작하면:

  • 중간에 고객이 연락을 끊어도 아무 대가 없이 시간을 잃음
  • 작업 방향 변경이 무한정 반복될 수 있음
  • 고객의 진지도 확인 불가
  • 고객의 진지도 확인됨
  • 방향 변경 요청 시 "착수금 범위 내에서" 제한 가능
  • 납품 거절 시 착수금은 보존

지급 기한 설정

상황권장 기한
개인 고객납품 후 7일 이내
중소기업납품 후 14일 이내
대기업·기관납품 후 30일 이내 (관행)

> "잔금은 최종 납품 확인 후 [14]일 이내에 을의 계좌로 송금한다."

지급이 늦어질 경우를 대비:
> "대금 지급이 기한을 초과하는 경우 연체일수에 대해 연 [12~15]%의 이자를 가산한다."

→ 없어도 되지만 있으면 지연 방지 효과

> B2B 서비스 위탁계약서의 7대 핵심 조항

위탁계약서 vs NDA

NDA위탁계약서
목적비밀 보호거래 조건 확정

조항 1: 계약 당사자

```
갑 (발주자): [고객사명] / 대표자: [이름] / 주소: [주소]
을 (수주자): [내 회사명] / 대표자: [이름] / 주소: [주소]
```

  • 개인이 아닌 사업자명으로 (사업자등록증 기준)
  • 주민등록번호 대신 사업자등록번호

조항 2: 용역 범위

  • 포함되는 것: 구체적으로
  • 포함되지 않는 것: 구체적으로 (더 중요)

> "을은 아래의 서비스를 제공한다:
> 포함: 월간 온라인 매거진 기획(1회), 콘텐츠 10꼭지, 편집디자인, 웹 발행
> 미포함: SNS 홍보 대행, 인쇄물 제작, 영상 콘텐츠, 10꼭지 초과 원고"

조항 3: 대금·지급 조건

  1. 총 금액 + VAT 별도 명시
  2. 지급 방식 (선금 비율·잔금 시점)
  3. 지급 기한 (납품 후 XX일 이내)
시점비율금액

조항 4: 납품·검수 조건

  • 납품 기한
  • 납품 방식 (이메일/링크/파일)
  • 검수 기간 (고객이 확인할 수 있는 기간)
  • 수정 횟수 상한

> "을은 매월 [XX]일까지 납품한다.

> 수정은 [2]회까지 무상으로 제공하며, 초과 시 건당 별도 청구한다."

조항 5: 지적재산권 귀속

  • 결과물 권리: 고객에게 이전? 공동 소유?
  • 방법론·도구: 내가 보유
  • 이전 시점: 대금 완납 후

> "결과물의 저작권은 대금 완납 시 갑에게 귀속된다.
> 단, 을이 보유한 방법론·템플릿·자체 도구는 을의 소유이며 이전 대상에서 제외된다."

조항 6: 계약 해지 조건

해지 사유처리

상호 합의 해지진행 완료분 대금 정산 후 종결
불가항력이행 유예 또는 협의 해지
